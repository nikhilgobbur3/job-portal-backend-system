# Job Portal Backend (Spring Boot)

This project provides a Job Portal REST API using Java, Spring Boot, Spring Security (JWT), MySQL, Spring Data JPA.

Quick start:

1. Create MySQL database `job_portal_db` and update `application.properties` with credentials.
2. Build & run:

```bash
mvn clean package
java -jar target/job-portal-0.0.1-SNAPSHOT.jar
```

API docs: http://localhost:8080/swagger-ui.html or Springdoc at `/swagger-ui/index.html` once running.
